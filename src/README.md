# Source Directory

This directory contains all source code and assets for the NeoConvertXlsx project.

## Structure

```
src/
├── css/           # Stylesheets
├── js/            # JavaScript files
└── assets/        # Static assets (images, icons, etc.)
```

## Directories

-   **css/** - Contains all CSS files for styling and theming
-   **js/** - Contains all JavaScript files for application logic
-   **assets/** - Contains static assets like images and media files

Each subdirectory has its own README with specific information about its contents.
