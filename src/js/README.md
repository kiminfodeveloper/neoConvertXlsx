# JavaScript Directory

This directory contains all JavaScript files for the NeoConvertXlsx project.

## Files

-   `main.js` - Main JavaScript file containing all application logic, event handlers, and functionality

## Functionality

The JavaScript handles:

-   Data processing and conversion logic
-   User interface interactions
-   File handling and validation
-   Theme management (light/dark mode)
-   Page navigation
-   Modal operations
-   Real-time feedback and logging

## Architecture

The code is organized into:

-   Configuration objects
-   Helper functions
-   Event listeners
-   Processing functions
-   UI update functions
